== About

A simple rails app that allows a user to sign in with a twitter account and send a tweet.
Uses MongoDB & Mongoid, Omniauth for authentication, and 'Twitter' gem for tweeting.


== Setup

Go to http://dev.twitter.com to get your application's Keys. Replace the X's in 
* config/intialisers/omniauth.rb
* controllers/application_controller.rb 
with your respective keys.
